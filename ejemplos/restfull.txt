1. Buenas practicas en las urls
    Servidor:http://localhost:8080
    /pizza->No

    1.1 Debo de pluralizar el dominio
        /pizzas
    1.2 Mi url no debe de exponer acciones

        /pizzas/create(no)-> /pizzas verbo http POST
        /pizzas/update(no)   /pizzas verbo http PUT
    1.3 No exponer formatos en mi url
        /pizzas.json 
        /pizzas.xml
        
        https://developer.mozilla.org/en-US/docs/Web/HTTP/Guides/MIME_types/Common_types

        accept:"application/json"
        content-type:"application/json"
    1.4 versionado de mi api
        /v1/pizzas
